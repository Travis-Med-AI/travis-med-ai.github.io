# How To Contribute

We want contributing to Travis Med AI to be simple and enjoyable.
All contributions are welcome, including issues, new docs as well as updates and tweaks, blog posts, workshops, and more.

See our [contributing models](/contributing/contributing-models) section for more information on how to contribute new models.
