# How To Contribute

We want contributing to Travis Med AI to be simple and enjoyable.
All contributions are welcome, including issues, new docs as well as updates and tweaks, blog posts, workshops, and more.

# Machine Learning Models

All machine learning models are managed [here](https://github.com/Travis-Med-AI/med-ai-models)

Documentation related to the models can be found [here](https://travis-med-ai.github.io/med-ai-models/)
